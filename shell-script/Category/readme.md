# API CATEGORY

La API de categoría no posee dependencia, es totalmente independiente, posee una programación tipo "Árbol Binario" donde dentro de la misma ramificación toma los datos preguntando si el valor "hasChildren" es verdadero, de ser así, se obtiene el padre, los hijos y los hermanos dentro de una ramificación de categorías, esta API depende de sólo una ejecución.

- Dentro del directorio shell-script se busca el directorio categoría y se ejecutar de esta forma: **sh category.sh**
